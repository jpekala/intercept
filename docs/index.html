Quick Start

Get up and running in minutes

Supported Platforms: Officially tested on Debian and Ubuntu. Partial support for macOS. Other distributions have not been fully tested.

Standard Installation

git clone https://github.com/smittix/intercept.git
cd intercept
./setup.sh
sudo -E venv/bin/python intercept.py

Requires Python 3.9+ and RTL-SDR drivers

Docker

git clone https://github.com/smittix/intercept.git
cd intercept
docker compose up -d

Requires privileged mode for USB SDR access

After starting, open http://localhost:5050 in your browser.

Default credentials: admin / admin
